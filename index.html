Export & SharePoint Steps

  1. Click Download Populated XLSM to get the file with AP Tracker and Net Equip filled in.
  2. Open it in Excel — the AP Name and Description columns will auto-calculate from the data rows.
  3. Upload to your SharePoint project folder as normal.
  4. The Site Data tab will still need SSID passwords and PM/designer info filled in if required for your process.
The exported XLSX pre-fills: Site Data (site name, CLLI, vendor, vertical, vSZ zone, AP count), AP Tracker (serial, raw MAC in col K, model, AP number, building, floor, installed area), and Net Equip Tracker (device type, model, serial, raw MAC, building, floor, device number, location). Formula columns (AP Name, Description, formatted MAC) recalculate automatically when opened in Excel.
